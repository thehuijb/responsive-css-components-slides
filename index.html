Rem vs Em

An EM is a unit of typography, equal to the currently specified point-size

- Wikipedia

Em

h1 {
	font-size: 20px; /* 1em = 20px */
	margin: 1em; /* 20px */
}

p {
  font-size: 16px; /* 1em = 16px */
	margin: 1em; /* 16px */
}

Rem

A REM is a unit of typography
equal to the root font-size

Rem

html {
	font-size: 16px;
}

h1 {
	font-size: 20px; /* 1rem = 16px */
	margin: 1em; /* 16px */
}

p {
  font-size: 16px; /* 1rem = 16px */
	margin: 1em; /* 16px */
}

Rem or Em?

  • Size in em if the property scales according to its font-size
  • Size everything else in rem.

Rem or Em?

.header{
	font-size: 1rem;
	padding: 0.5em 0.75em;
}

.header--large{
	font-size: 2rem;
}

Rem or Em?

.header{
	font-size: 1rem;
	/* Em for top/bottom padding.
	 * Rem for left/right padding
	 */
	padding: 0.5em 0.75rem;
}

.header--large{
	font-size: 2rem;
}

Rem or Em?

.header {
	font-size: 1rem;
	padding: 0.75rem 0.75rem;
}

.header--large {
	font-size: 2rem;
	// Correcting for Vertical Rhythm
	padding: 1.25rem 0.75rem;
}

Scaling Summary

  • Determine component area maps
  • Determine breakpoints and changes in styles
  • Determine units for CSS properties
  • Handle complex variations with mixins or element queries
  • Don't over engineer
